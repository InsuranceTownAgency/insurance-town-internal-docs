Ricochet Lead Scrub Guide

Permanent weekly responsibility. Pull reports. Scrub for DNC. Categorize by source. Upload clean lists back to Ricochet. Flag invalid numbers on AZ lead cards. Weekly rolling 30-day window.
Overview and Rules
Read this tab first every week before starting the scrub. Never skip steps. Never contact any lead yourself.
Critical Rules

Never contact any lead yourself. Your job is to scrub, flag, and report only.
Never delete any record in AZ. Flag with a note only.
Never upload an unscrubbed list to Ricochet. Always scrub first.
Always use Eva's admin credentials. Never your own login. Never share credentials.
Report to Eva before uploading anything back to Ricochet.

Your weekly scrub flow in 4 steps:
Step 1: Pull lead reports from Ricochet (Deedma portal)
Step 2: Run each list through RealValidation scrub
Step 3: Upload clean categorized lists back to Ricochet
Step 4: Flag DNC and invalid numbers on AZ lead cards
Frequency: Weekly. Rolling 30-day window. Every Monday at the start of your shift.
Lead sources you scrub (4 total):
1. Facebook Meta Ads: Home and Commercial
2. Internet lead companies: Home and Auto purchased
3. Recycled Ricochet leads (re-scrub every cycle)
4. CSLB and LinkedIn researcher leads (Jaffer delivers these as Excel files — add to scrub queue when received)
TCPA Compliance — Never Skip This:
Facebook leads: compliant for 90 days from opt-in date. After 90 days = email only, no SMS, no calls.
Purchased internet leads: compliant for 30 days from purchase date.
Recycled leads: re-scrub required every cycle before re-uploading.
If you find a lead over its compliance window, flag it in your EOD with the lead name and date. Do not upload it.

Step 1: Pull Reports from Ricochet
Log into the Ricochet admin portal (Deedma) and download each lead source as a separate CSV file. Keep every source in its own file. Never combine sources.
1
Log into Ricochet Admin Portal
Go to reports.ricochet.me/login. Use Eva's admin credentials from 1Password (search: Ricochet Admin). Do not use the regular Ricochet login.
https://reports.ricochet.me/login
2
Pull Rolling 30-Day Lead Report
Set the date range to the last 30 days from today. Pull one report per lead source. Download as CSV each time. Name each file exactly as shown:

Facebook Home leads: FB_Home_[YYYY-MM-DD].csv
Facebook Commercial leads: FB_Commercial_[YYYY-MM-DD].csv
Internet Home leads: Purchased_Home_[YYYY-MM-DD].csv
Internet Auto leads: Purchased_Auto_[YYYY-MM-DD].csv
Recycled leads: Recycled_[YYYY-MM-DD].csv
Replace [YYYY-MM-DD] with today's date. Example: FB_Home_2026-06-29.csv
3
Check TCPA Compliance Dates
Before scrubbing, check the opt-in or purchase date on each lead.

Facebook leads: Remove any lead where opt-in date is more than 90 days ago. Flag these in your EOD.
Purchased leads: Remove any lead where purchase date is more than 30 days ago. Flag these in your EOD.
Do not delete these from AZ. Just remove them from the upload CSV and note them in your EOD report.
Step 2: Run RealValidation Scrub
Upload each CSV file separately to RealValidation. Run DNC and phone validation on each one. Download the clean output. Never skip this step before uploading to Ricochet.
1
Log into RealValidation
Go to realvalidation.com. Use Eva's account credentials from 1Password (search: RealValidation). The Authorization Token is in the Notes section of that 1Password entry.
https://www.realvalidation.com
2
Upload Each CSV Separately
Upload one CSV at a time. Start with Facebook Home, then Facebook Commercial, then Purchased Home, then Purchased Auto, then Recycled. Run the full DNC and phone validation scrub on each file. Wait for results before moving to the next file.
Never combine files before scrubbing. Each source must be scrubbed separately so the categorization stays clean for Kyle's routing automation.
3
Download Clean Output
Download the clean output file for each source. RealValidation will show which numbers are DNC, invalid, or clean. Save the clean output with this naming format:

CLEAN_FB_Home_[YYYY-MM-DD].csv
CLEAN_FB_Commercial_[YYYY-MM-DD].csv
CLEAN_Purchased_Home_[YYYY-MM-DD].csv
CLEAN_Purchased_Auto_[YYYY-MM-DD].csv
CLEAN_Recycled_[YYYY-MM-DD].csv
4
Save the DNC and Invalid List Separately
From each scrub result, save the flagged numbers (DNC and invalid) as a separate file. Name it:

FLAGGED_[Source]_[YYYY-MM-DD].csv

You will use this in Step 4 to flag the AZ lead cards.
Step 3: Upload Clean Lists to Ricochet
Upload each clean categorized file back to Ricochet in the correct campaign. Never mix sources. Always report to Eva before activating any list.
Before You Upload Anything

Send Eva a Slack message with a summary of what you scrubbed and what you found. Wait for Eva's reply before uploading. Never upload without Eva's green light.

1
Send Eva the Scrub Summary First
Before uploading anything, message Eva in Slack with:

Total leads per source reviewed
How many removed for TCPA compliance
How many flagged as DNC
How many flagged as invalid phone
How many clean and ready to upload per source
Wait for Eva to reply "go ahead" before proceeding to the upload step.
2
Upload Each Clean File to the Correct Ricochet Campaign
Log back into Ricochet. Upload each clean CSV to its matching campaign:

CLEAN_FB_Home → Campaign 1: New Home/Auto leads
CLEAN_FB_Commercial → Campaign 2: Commercial CSLB and LinkedIn
CLEAN_Purchased_Home → Campaign 1: New Home/Auto leads
CLEAN_Purchased_Auto → Campaign 1: New Home/Auto leads
CLEAN_Recycled → correct campaign based on lead type
If you are unsure which campaign a file belongs to, stop and message Eva before uploading.
Step 4: Flag DNC and Invalid Numbers in AZ
For every number that came back as DNC or invalid in the scrub, find that lead card in AgencyZoom and add a note. Never delete anything. Flag only.
1
Open Your Flagged CSV File
Open the FLAGGED_[Source]_[Date].csv file you saved in Step 2. This has every number that came back as DNC or invalid. Go through each one.
2
Find the Lead Card in AgencyZoom
Search for the lead by phone number or name in AZ. Open the lead card.
3
Add a Note on the Lead Card
Add this exact note format on the AZ lead card:

For DNC: "RealValidation: DNC — do not contact. Flagged [today's date]."
For invalid phone: "RealValidation: Invalid phone number. Flagged [today's date]."
Do not close, delete, or move the card. Do not tag the card. Just add the note. Eva decides next steps.
4
Log Every Flagged Lead in Your EOD
In your EOD report, list every lead you flagged in AZ with: lead name, phone number, source, and reason (DNC or invalid). See the EOD tab for the exact format.
